# Gerador de curiosidades e quiz com moviepy
## Instalação

1. Clone este repositório
2. execute o script com:
``./install.sh``
3. Pode acontecer de ser necessário dar permissão de execução ao install.sh: `chmod +x install.sh`
4. Para criar um novo quiz basta clicar na tela com o botão direito e escolher, novo documento quiz.
5. Descompacte **quiz.zip**, utilize o prompt fornecido para gerar novas perguntas e respostas e adicioná-los a dados.csv
6. Abra a pasta no terminal e execute `quiz`.
7. Para gerar curiosidades execute `shorts`.
